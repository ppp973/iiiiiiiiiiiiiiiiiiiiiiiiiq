# Study IQ Telegram Bot

## Features
- Extract Study IQ course videos and PDFs
- Support for multiple batch downloads
- Token save feature for quick access
- Detailed statistics of extracted content

## Setup
1. Get CHANNEL_ID from @getidsbot
2. Update CHANNEL_ID in config.py
3. Deploy on Suno Render

## Commands
- /start - Welcome message
- /help - Help guide
- /iq - Start extraction
- /about - Bot info
